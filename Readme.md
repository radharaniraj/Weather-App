 Weather App is a premium app that puts your current conditions and a short-term forecast front  when you open it.<br>
 It gives the information about latitude,longitude,raining probability percentage ,temperature and also so many features of any desired location.<br>
 We put a valid input in our terminal for which we have to determine the current condition and start our server at port 3000.
 <br>
 Here i used a restful API of Darksky ,that are globally available.
